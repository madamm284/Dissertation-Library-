# Dissertation Library CLI (Python)

A simple command-line application for managing dissertations.
Built with Python using object-oriented programming principles.

# Features:

# Add new dissertations
-Search by:
Title
Author
Supervisor
# View all dissertations
-Update completion status
-Edit dissertation details
- Remove dissertations
- Case-insensitive search
# Technologies Used
Python 3
Object-Oriented Programming (OOP)
Lists & filtering (filter, lambda)
CLI (Command Line Interface)
# How It Works

#The application is built around two main classes:

-Dissertation

Represents a single dissertation with:

title
author
supervisor
completion status
DissertationLibrary

-Manages a collection of dissertations and provides methods to:

Add dissertations
Search (title, author, supervisor)
Update completion status
Remove dissertations
Edit dissertation details
Display all entries
